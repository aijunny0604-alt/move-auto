# 📸 무브 오토모티브 사진 전시관 업그레이드 완료

## ✨ 새로 추가된 기능

### 1. 🖼️ 라이트박스 기능
- **사진 클릭** → 전체화면으로 확대
- **ESC 키** 또는 **닫기 버튼(X)**으로 닫기
- **배경 클릭**으로도 닫기 가능
- **화살표 키**로 이전/다음 사진 이동 (gallery.html에서)

### 2. 📅 연도별 전시관 (gallery.html)
- **2024년**: 무브 오토모티브의 시작
  - 파왕쿱 V1 첫 공개
  - AMF 국제 모터 페스티벌 2위
  - 첫 스폰서 계약 체결
  - 오토살롱 튜닝 페스티벌
  - 연말 감사 팬미팅

- **2025년**: 일본 진출 & 국내 대회 우승
  - 일본 D1 LIGHTS 첫 출전
  - 부산 국제 모터쇼 전시
  - KDGP 챔피언십 우승
  - 유튜브 10만 구독자 달성

- **2026년**: D1 LIGHTS 시즌
  - 개막전 준비
  - 서킷 트레이닝
  - 팬 미팅 & 사인회

### 3. 📝 상세한 사진 설명
각 사진마다 포함된 정보:
- **날짜**: 행사 일자
- **제목**: 이벤트명
- **설명**: 행사의 의미와 하이라이트

### 4. 🎨 프리미엄 전시관 디자인
- **다크 테마**: 고급스러운 검정 배경
- **레드 액센트**: 무브 오토모티브 브랜드 컬러
- **카드 호버 효과**: 마우스 오버시 부드러운 애니메이션
- **반응형 디자인**: 모바일/태블릿 완벽 지원

## 🚀 사용 방법

### index.html에 적용하기
메인 페이지(`index.html`)의 `</body>` 태그 바로 위에 다음 코드를 추가하세요:

```html
<script src="gallery-button.js"></script>
```

### 파일 구조
```
move-auto/
├── index.html              (메인 페이지)
├── gallery.html            (사진 전시관 - 새로 생성)
├── gallery-button.js       (자동 버튼 추가 스크립트)
└── GALLERY_GUIDE.md       (이 파일)
```

## 🎯 스폰서 홍보 효과

### 1. 전문성 강조
- 연도별 성장 스토리 시각화
- 주요 대회 성과 부각
- 전문 모터스포츠 팀 이미지

### 2. 브랜드 노출
- 각 이벤트마다 상세 설명
- 고화질 사진으로 차량 디테일 강조
- 스폰서 로고가 담긴 사진 전시

### 3. 미디어 가치
- 12개 이상의 주요 행사 사진
- 연도별 정리로 성장세 입증
- SNS 공유 가능한 고급 레이아웃

## 💡 추가 개선 아이디어

### 실제 사진 추가하기
`gallery.html` 파일에서 각 카드의 `<span class="image-placeholder">📷</span>` 부분을:

```html
<img src="images/2024-pawang-v1.jpg" alt="파왕쿱 V1">
```

이렇게 실제 이미지로 교체하세요.

## 🎬 테스트 방법

1. **브라우저에서 gallery.html 열기**
2. **사진 카드 클릭** → 라이트박스 확인
3. **화살표 키** → 이미지 전환 테스트
4. **모바일 뷰** → 반응형 디자인 확인
